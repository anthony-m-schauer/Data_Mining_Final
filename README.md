# S&P 500 Data Mining Project

## Setup
1. Clone the repo:
   git clone https://github.com/anthony-m-schauer/Data_Mining_Final.git
2. Install dependencies:
   pip install -r requirements.txt

## Usage
1. Open VS Code in the Data_Mining_Final folder.
2. Edit Python scripts or the main notebook.
3. Run the notebook to execute the pipeline.

## Modules
- data_pipeline.py: download data, compute returns, normalize
- similarity.py: Pearson and cosine similarity
- clustering.py: K-Means, hierarchical, optional DBSCAN
- visuals.py: heatmaps, PCA scatter, dendrograms
